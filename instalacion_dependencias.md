# **Instalación de Dependencias**

---

## **Entorno Virtual**

**Abrir terminal**

**Para instalar paquete, comando:**
```bash
pip install virtualenv
```

**Para activar entorno virtual:**
```bash
.\.venv\Scripts\activate
```

**Para desactivar entorno virtual:**
```bash
deactivate
```

---

## **Instalación de paquetes**

**Desde allí instalo los nuevos paquetes:**

### **Para Flask**
```bash
pip install flask
```

### **Para OWASP ZAP**
```bash
pip install zaproxy
```

### **Para Gemini**
```bash
pip install langchain-google-genai
```
### **Para Flask-cors **
```bash
pip install flask-cors
```

### **Para requests **
```bash
pip install requests
```

### **Para beautifulsoup4 **
```bash
pip install beautifulsoup4
```
